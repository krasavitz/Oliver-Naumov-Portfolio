Reflect

SUNY Albany CINF 362

Fall 2023

Context

"Reflect" is a digital journaling platform I created as the final project for SUNY Albany's CINF 362 course on client-based web development. The course focused on practical web development technologies and best practices. I chose to create an introspective journaling platform that encourages daily reflection through guided prompts.

Problem

The challenge was to create a guided journaling experience that makes self-reflection more accessible and engaging. The solution needed to address the difficulty of starting and maintaining a journaling habit, the challenge of coming up with meaningful reflection topics, the desire for a structured yet flexible journaling approach, and the need for a simple, distraction-free writing environment.

Role

I was responsible for all aspects of this academic project, including full-stack web development, user experience design, database architecture, frontend implementation, and content strategy for daily prompts.

Impact

I created "Reflect" as the final project for SUNY Albany's CINF 362 course on client-based web development. The course focused on practical web development technologies and best practices. I chose to create an introspective journaling platform that encourages daily reflection through guided prompts. The project successfully demonstrated my proficiency in full-stack web development, showcased my ability to create a complete web application, applied course concepts in a practical context, created a functional platform for daily reflection, and received positive feedback during the course presentation.
